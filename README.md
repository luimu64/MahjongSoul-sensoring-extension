# MahjongSoulless
Simple browser extension that blocks Mahjong Soul from loading the character portraits and loading screen wallpapers.

## Installation
Only works in chromium derivate browsers.

1. Download the repo as a zip
2. Unzip it anywhere
3. Go to your browser extensions page
4. Enable developer mode
5. Click "load unpacked" and select the folder you unpacked the zip
6. Go to [Mahjong Soul](https://mahjongsoul.game.yo-star.com) and enjoy your anime-girl-less mahjong experience
